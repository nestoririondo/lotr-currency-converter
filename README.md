# LOTR Currency Exchange

A SwiftUI app for converting between different currencies from the Lord of the Rings universe.

## Features

- Convert between 5 different LOTR currencies:
  - Copper Penny
  - Silver Penny
  - Gold Penny
  - Silver Piece
  - Gold Piece
- Real-time bidirectional conversion
- Interactive currency selection
- Exchange rate information view

## Technologies

- SwiftUI
- Swift 6.0+
- iOS 17.0+

## Getting Started

1. Clone the repository
2. Open `LOTR.xcodeproj` in Xcode
3. Build and run on simulator or device
